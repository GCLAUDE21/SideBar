# SIDEBAR

Projet interactif réalisé dans le cadre de ma formation Développeur Web.

**Demo live**
[Voir la page](https://gclaude21.github.io/SideBar/)

## Objectif

Créer une sidebar interactive en JavaScript natif : un bouton hamburger ouvre et ferme la sidebar, avec un effet d'ombre sur le contenu principal et une fermeture au clic en dehors.

## Compétences travaillées

- Manipulation du DOM avec JavaScript natif
- Gestion des événements (click, stopPropagation)
- Toggle de classes CSS via JavaScript
- Transformation CSS avec `rotate` pour l'effet hamburger vers croix

## Stack technique

JavaScript natif. HTML natif. CSS natif. Aucun framework ou bibliothèque externe.

## Structure du projet

```
sidebar/
├── index.html
├── style.css
└── index.js
```
